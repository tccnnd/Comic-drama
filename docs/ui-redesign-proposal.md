# UI 重构方案

## 当前问题诊断

### 1. 信息架构混乱
- **8 个顶部 Tab** 太多：项目总览、项目设置、剧本识别、角色与声线、外部引擎、分镜工作台、当前分镜、合成导出
- 用户不知道该从哪个 Tab 开始，工作流不连贯
- "分镜工作台"和"当前分镜"功能重叠

### 2. 工作流断裂
- 创建项目后，用户需要手动跳转多个 Tab 才能完成一个完整流程
- 没有明确的"下一步"引导
- 生成状态散落在各处，没有统一的进度视图

### 3. 视觉层次不清
- 所有面板视觉权重相同，没有主次
- 侧边栏占用过多空间（项目列表 + 新建项目 + 运行状态）
- 分镜卡片信息密度太高，关键信息（画面预览）反而不突出

### 4. 生产效率低
- 没有批量操作（批量重绘、批量审片）
- 候选对比需要来回切换
- 时间线太小，不能直观操作

---

## 重构方案：三区布局 + 阶段引导

### 核心理念
参考 Toonflow 的无限画布 + LumenX 的阶段式 SOP，重构为：

```
┌─────────────────────────────────────────────────────────────┐
│ 顶栏：项目名 + 阶段指示器 + 全局操作                          │
├────────┬────────────────────────────────────┬───────────────┤
│        │                                    │               │
│ 导航栏  │         主工作区                    │   属性面板     │
│        │                                    │               │
│ 项目   │  (根据当前阶段显示不同内容)           │  (选中对象的   │
│ 阶段   │                                    │   详细编辑)    │
│ 场景   │  阶段1: 剧本画布                    │               │
│ 列表   │  阶段2: 角色资产                    │               │
│        │  阶段3: 分镜时间线                   │               │
│        │  阶段4: 审片导出                    │               │
│        │                                    │               │
├────────┴────────────────────────────────────┴───────────────┤
│ 底栏：时间线（始终可见）+ 播放控制                              │
└─────────────────────────────────────────────────────────────┘
```

### Tab 精简：8 → 4 个阶段

| 原 Tab | 新阶段 | 说明 |
|--------|--------|------|
| 项目总览 + 项目设置 | **① 策划** | 项目信息、风格选择、故事输入 |
| 剧本识别 + 角色与声线 | **② 资产** | 角色卡片、声线、参考图、风格 |
| 分镜工作台 + 当前分镜 | **③ 分镜** | 时间线、分镜编辑、画面预览 |
| 合成导出 + 外部引擎 | **④ 出片** | 批量生成、审片、导出 |

## 视觉设计改进

### 配色保持暗色主题，但增加层次

```css
/* 新增变量 */
--surface-elevated: #1a2332;  /* 弹出面板 */
--accent-glow: rgba(103, 183, 255, 0.15);  /* 选中态光晕 */
--progress-track: #1e2938;
--progress-fill: linear-gradient(90deg, #67b7ff, #54d19c);
```

### 关键视觉改进
1. **分镜卡片** — 缩略图占 70% 面积，文字信息精简为一行
2. **状态指示** — 用颜色圆点而不是文字（✅🟡🔴⚪）
3. **预览区** — 圆角 + 微阴影 + 16:9 或 9:16 固定比例
4. **按钮层级** — 主操作（蓝色实心）、次操作（边框）、危险（红色）
5. **动画** — Tab 切换淡入淡出、卡片 hover 微上浮

## 交互改进

### 快捷键
| 快捷键 | 操作 |
|--------|------|
| `1-4` | 切换阶段 |
| `←/→` | 上/下一个分镜 |
| `Space` | 播放/暂停预览 |
| `R` | 重新生成当前分镜 |
| `S` | 保存 |
| `E` | 导出 |

### 拖拽操作
- 分镜缩略图条：拖拽重排场景顺序
- 时间线：拖拽调整时长
- 角色卡片：拖拽到场景分配角色

### 右键菜单
- 分镜卡片：重绘图 / 重配音 / 重合成 / 拆分 / 合并 / 删除
- 角色卡片：重新生成 / 编辑 / 删除
- 时间线片段：调整时长 / 设置转场 / 锁定

## 实施优先级

### Phase 1（立即）— 不改架构，只调布局
1. Tab 从 8 合并为 4
2. 分镜工作台的预览区放大
3. 侧边栏精简
4. 底部时间线始终可见

### Phase 2（短期）— 组件重构
5. 分镜卡片重新设计（大缩略图）
6. 属性面板独立出来（右侧固定）
7. 批量操作工具栏
8. 快捷键支持

### Phase 3（中期）— 交互升级
9. 拖拽重排
10. 候选对比视图
11. 实时预览播放
12. 导演审片工作流
